# i2b2 r plugin package 

## prerequired 

- i2b2 1.7 and higher
- postgresql database
- linux (debian/ubunu prefered)
- opencpu
- RPostgres package
  - libpq (postgresql development library)

## installation

- compile the package
- copy/edit the config file to a securized destination
